üìä How We Calculate Undervaluation

‚ö†Ô∏è Important: "Fair Value" is an algorithmic estimate, not an official financial metric or market price. It is based on performance data and should be used for analysis purposes only.

The Formula

Fair Value Estimate = Base Value √ó Performance Ratio √ó Age Multiplier √ó League Multiplier

Undervaluation % = (Fair Value Estimate - Market Value) / Market Value √ó 100

Performance Ratio

We use xG + xA per 90 minutes (expected goals + expected assists) compared to position averages:

  • Forwards: 0.55 xGI/90 average
  • Midfielders: 0.35 xGI/90 average
  • Defenders: 0.15 xGI/90 average

Age Curve

Player value peaks at 24-26. We apply multipliers:

  • 17-20: 1.4x (potential premium)
  • 21-24: 1.3x (peak value)
  • 25-28: 1.0x (prime years)
  • 29-32: 0.7x (decline)
  • 33+: 0.4x (veteran)

League Multipliers

  • Premier League: 2.0x
  • La Liga: 1.4x
  • Bundesliga: 1.3x
  • Serie A: 1.2x
  • Ligue 1: 1.1x

Data Sources & Freshness

xG/xA: Sourced from Understat. Data is updated via manual script execution.

Market Values: Algorithmic estimates based on performance data and league context.

‚è±Ô∏è Data Freshness: Check the timestamp shown in the app header. Data reflects the last time the update script was run.

Limitations

  • Fair Values are estimates, not official valuations
  • No injury history factored in
  • Contract length not included
  • Defensive metrics limited
  • Sample size varies by minutes played
  • Data freshness depends on manual script updates
